# Hướng dẫn sử dụng MyShell (Updated 06/11/2025)
## Một số lệnh:
### 1.Mở tiến trình ở chế độ forebackground:
Sử dụng lệnh "fopen". Khi mở ở chế độ này, shell sẽ tạm dừng đến khi tiến trình kết thúc.
#### Cú pháp:
```bash
fopen <tien_trinh_can_mo>
```
Ví dụ: fopen notepad
### 2.Mở tiến trình ở chế độ background:
Sử dụng lệnh "bopen". Khi mở ở chế độ này, shell sẽ hoạt động đồng thời với tiến trình.
#### Cú pháp:
```bash
bopen <tien_trinh_can_mo>
```
Ví dụ: bopen notepad

### 3. In ra danh sách tiến trình:
Sử dụng lệnh "list"

### 4. Hủy tiến trình:
Sử dụng lệnh "remove". Sau đó chọn tiến trình cần hủy theo số thứ tự của tiến trình đó.

### 5. Tạm dừng và tiếp tục tiến trình:
Sử dụng lệnh "stop" và "resume". Sau đó chọn tiền trình mình muốn.
