Mở tiến trình foreground mode: 
-Lệnh fopen + tiến trình cần mở
-VD: fopen notepad
Mở tiến trình background mode:
-Lệnh bopen + tiến trình cần mở
-VD: bopen mspaint
Lưu ý: Chỉ một số tiến trình hỗ trợ win32 mới dùng được, như msapint hoặc notepad. Còn ví dụ như calc có thể mở được nma vẫn chưa fix lỗi chạy foreground.
